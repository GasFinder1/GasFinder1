# Gas Finder⛽

Um projeto desenvolvido para auxiliar usuários a encontrar o melhor custo-benefício na hora de abastecer o seu veículo.


## Funcionalidades

- Busca de preços de combustível através da agência nacional do petróleo; 
- Acompanhar postos através de um mapa;
- Traças rotas até um determinado posto;
- Cálculo de rendimento entre etanol e gasolina;
- Favoritar postos;
- Sugerir preços para postos que não possuem preço cadastrado na ANP.


## Stack utilizada

**Front-end:** React, Bootstrap, Google Maps API

**Back-end:** Node.js, Express, XLSX

**Database:** MySQL
